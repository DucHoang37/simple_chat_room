# Simple_chat_room

-Cái này là mình tự tìm hiểu trên mạng và sửa lại một số chỗ để nó "thuần Việt" hơn
-Cách sử dụng:
    +B1: chạy file server.py
    +B2: chạy file user.py, có thể hỗ trợ lên đến hơn 20user

-Một số lưu ý nhỏ:
    +Ứng dụng có hỗ trợ tiếng Việt khi mọi người trò chuyện
    +Các bạn có thể thêm một số tính năng cho đoạn chat (vd: kick/ban user,...)
